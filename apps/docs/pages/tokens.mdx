# Design Tokens

Les tokens sont définis dans `packages/tokens/src/tokens.json` et exportés en CSS variables via `packages/tokens/dist/vars.css`.

```css
:root[data-brand="muzipics"] {
  --brand: #ff2d55;
  --brand-on: #1a0b0f;
}
```

Tailwind preset partagé : `@avnir/tokens/tailwind-preset.cjs`.

Pour changer de marque globalement, définir sur `<html>` :

```html
<html data-brand="muzibase"></html>
```
